## 🎯 ПСИХОЛОГИЯ ЦЕН:

✅ **Все цены на 9** → максимальный психологический эффект  
✅ **Второй пакет** → выглядит в 2 раза дешевле (€7.99 vs €11.97 по цене первого)  
✅ **Третий пакет** → якорь (дорогой, но заставляет второй выглядеть выгодным)  
✅ **€0.0266 и €0.0226** → цифры с 33 и 23 на конце (подсознательно дешевле, чем 40 или 50)

## 🔑 STRIPE КОНСТАНТЫ ДЛЯ КОДА

Когда создашь цены в Stripe → вставь эти Price ID:

```python
# bot/stripe_handler.py — АКТУАЛЬНЫЕ PRICE ID'Ы

STRIPE_PRICES = {
    "small": {
        "price_id": "price_1TLetkLhGc6RmyWQCzBtrlmH",
        "posts": 100,
        "price_eur": 3.99,
        "price_per_post": 0.0399,
    },
    "medium": {
        "price_id": "price_1TLetkLhGc6RmyWQLcHxycOa",
        "posts": 300,
        "price_eur": 7.99,
        "price_per_post": 0.0266,
        "discount_percent": 33,  # vs small
        "discount_value_eur": 3.98,  # €11.97 - €7.99
    },
    "large": {
        "price_id": "price_1TLetkLhGc6RmyWQ8cvtIsTI",
        "posts": 1500,
        "price_eur": 33.99,
        "price_per_post": 0.0226,
        "discount_percent": 73,  # vs small
        "discount_value_eur": 25.86,  # €59.85 - €33.99
    },
}

# Для расчёта скидок
BASE_PRICE_PER_POST = 0.0399  # цена за пост в малом пакете
```

| Метрика | Значение | Почему |

| €0.0226 | третий | Цифра 26 на конце еще дешевле |

## 🎯 СТРАТЕГИЯ ПРОДАЖ

1. **Пакет 1 (€3.99)** → входная цена для пробующих
2. **Пакет 2 (€7.99)** → ⭐ ОСНОВНОЙ выбор (выглядит очень выгодным)
3. **Пакет 3 (€33.99)** → якорь (мало кто купит, но делает остальные выгодными)

**Ожидаемое распределение:**
- Пакет 1: 10% клиентов (новички)
- Пакет 2: 85% клиентов (основная масса)
- Пакет 3: 5% клиентов (профи)

**Средняя выручка на клиента:**
```
(0.10 × €3.99) + (0.85 × €7.99) + (0.05 × €33.99) = €8.19
```

При 50 клиентах в месяц → **€409.50 выручки**
