# Дизайн системы уведомлений (Фаза 6)

## 📲 Какие типы уведомлений нужны?

### 1. **Баланс заканчивается** ⚠️
- **Когда:** Баланс ≤ порога (default 30)
- **Как часто:** 1 раз в день
- **Важность:** Высокая
- **Пример:**
```
⚠️ БАЛАНС ЗАКАНЧИВАЕТСЯ

Осталось: 20 постов
Можете опубликовать: 2 рассылки по 10 групп

[💳 Купить] [Позже]
```

---

### 2. **Рассылка завершена** ✅
- **Когда:** После завершения массовой рассылки
- **Как часто:** Каждый раз
- **Важность:** Средняя
- **Пример:**
```
✅ РАССЫЛКА ЗАВЕРШЕНА

Результаты:
├─ Отправлено: 8 из 10 групп
├─ Потрачено: 8 постов
├─ Баланс: 92 поста

⚠️ Забаны: @group1, @group2
```

---

### 3. **Группы добавлены в черный список** 🚫
- **Когда:** Бот обнаружил что группа забанила
- **Как часто:** При каждом новом забане
- **Важность:** Средняя
- **Пример:**
```
🚫 ГРУППЫ ЗАБЛОКИРОВАНЫ

Во время рассылки произошел забан:

❌ @blocked_group1 (ошибка: нет прав)
❌ @blocked_group2 (ошибка: забан юзера)

Эти группы отключены для рассылки.

[📋 Просмотреть все] [◀️ Назад]
```

---

### 4. **Расписание отключено (напоминание)** 📅
- **Когда:** Расписание отключено, а пользователь пытался что-то сделать
- **Как часто:** Один раз в день
- **Важность:** Средняя
- **Пример:**
```
📅 РАСПИСАНИЕ ОТКЛЮЧЕНО

Ваше автоматическое расписание отключено.
Рассылка не будет происходить по графику.

[⏰ Включить расписание] [Позже]
```

---

### 5. **Ошибка при попытке рассылки** ❌
- **Когда:** Рассылка не смогла запуститься (нет постов, групп, и т.д.)
- **Как часто:** При ошибке
- **Важность:** Средняя
- **Пример:**
```
❌ РАССЫЛКА НЕ СМОГЛА ЗАПУСТИТЬСЯ

Причина: Нет активных групп

Что делать:
1. Добавьте группы
2. Убедитесь что группы не в черном списке
3. Попробуйте тест

[🎯 Добавить группы] [🧪 Тест]
```

---

### 6. **Платеж успешен** 💳
- **Когда:** Stripe webhook подтвердил платеж
- **Как часто:** После оплаты
- **Важность:** Высокая
- **Пример:**
```
💳 ПЛАТЕЖ УСПЕШЕН

Вы купили: 600 постов / €20
Новый баланс: 750 постов

Спасибо! Начните рассылку!

[📢 Рассылка] [🎁 Другие пакеты]
```

---

### 7. **Платеж отклонен** ⚠️
- **Когда:** Карта отклонена или другая ошибка Stripe
- **Как часто:** При ошибке платежа
- **Важность:** Высокая
- **Пример:**
```
⚠️ ПЛАТЕЖ ОТКЛОНЕН

Ошибка: Недостаточно средств на карте

Попробуйте другую карту или способ оплаты.

[💳 Попробовать снова] [📞 Поддержка]
```

---

### 8. **Важное обновление бота** 🔔
- **Когда:** У вас есть новые функции или важные правила
- **Как часто:** Редко
- **Важность:** Высокая
- **Пример:**
```
🔔 ВАЖНОЕ ОБЪЯВЛЕНИЕ

Добавлена новая функция: Анализ результатов рассылки

Теперь вы сможете видеть:
✨ Статистику по каждой группе
✨ Когда лучше рассылать
✨ Популярность постов

[📖 Подробнее]
```

## 💻 Обработчики уведомлений

### 1. Отправка уведомления с проверкой

```python
async def send_notification(
    user_id: int,
    notification_type: str,  # "balance_low", "broadcast_completed", etc.
    text: str,
    buttons: list | None = None,
) -> bool:
    """
    Отправить уведомление если оно не отключено.
    Возвращает True если отправлено, False если отключено.
    """
    bm = scoped_broadcast_manager(user_id)
    
    # Проверить что уведомление включено
    prefs = bm.state.get("notifications", {}).get("preferences", {})
    
    if notification_type not in prefs:
        # Неизвестный тип - отправить по-умолчанию
        pass
    elif not prefs[notification_type].get("enabled", True):
        # Уведомление отключено
        return False
    
    # Отправить
    await bot.send_message(user_id, text, reply_markup=buttons)
    
    return True
```

### 2. Инициализация уведомлений при /start

```python
def init_notifications(self):
    """Инициализировать настройки уведомлений по-умолчанию"""
    if "notifications" not in self.state:
        self.state["notifications"] = {
            "enabled": True,
            "preferences": {
                "balance_low": {"enabled": True, "threshold": 30},
                "broadcast_completed": {"enabled": True},
                "groups_blocked": {"enabled": True},
                "schedule_disabled": {"enabled": True},
                "broadcast_error": {"enabled": True},
                "payment_success": {"enabled": True},  # Required
                "payment_failed": {"enabled": True},   # Required
                "bot_updates": {"enabled": True},
            },
            "last_notifications_sent": {},
        }
        self.save()
```
